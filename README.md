##  Quick Start

### Prerequisites
- Node.js **22+** (`nvm use 22`)
- Docker & Docker Compose
- Git

### 1. Clone & Install
```shell
npm install
```
### 2. Start PostgreSQL
```shell
docker compose up -d postgres
```
### 3. Database Setup
```shell
npx prisma db push 
npx prisma generate 
```
### 4. Development Server
```shell
npm run dev
```
**âœ… Server ready at `http://localhost:3000` | ğŸ“š Docs at `http://localhost:3000/docs`**
